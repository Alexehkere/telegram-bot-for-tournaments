# Telegram Bot for Tournaments  

---

## Описание
**Telegram-бот для организации и проведения турниров** — мощный инструмент для геймеров, киберспортсменов и сообществ.  
Позволяет создавать турниры, регистрировать участников, автоматически формировать сетки, вести счёт и объявлять победителей — **всё прямо в Telegram**.

Проект разработан в рамках **курсового проекта по разработке ботов и интеграции с внешними API**.

---

## Функции

| Функция | Описание |
|--------|--------|
| **Создание турниров** | Название, формат (1x1, 2x2, FFA), призовой фонд, дата |
| **Регистрация участников** | Кнопка «Участвовать», проверка дубликатов |
| **Авто-сетка** | Генерация bracket (Single Elimination, Double Elimination) |
| **Матчи и результаты** | Уведомления, ввод счёта, подтверждение |
| **Рейтинг игроков** | ELO-подобная система, статистика |
| **Админ-панель** | Управление турнирами, бан/разбан, рассылка |
| **Логирование** | Все действия в БД + уведомления |

---

## Технологии
| Технология | Назначение |
|-----------|-----------|
| **Python 3.11** | Основной язык |
| **aiogram 3.x** | Асинхронный фреймворк для Telegram Bot API |
| **PostgreSQL** | Хранение турниров, пользователей, матчей |
| **SQLAlchemy / asyncpg** | ORM и работа с БД |
| **Docker & docker-compose** | Контейнеризация и деплой |
| **python-dotenv** | Управление переменными окружения |
